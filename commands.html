♟ Where Are You ♟

Find players' locations easily.

Commands for everyone:

  • /whereareyou - Shows all commands for Where Are You.
  • /whereareyou version - Shows the plugin version.
  • You can also use /whereareyou as /wru.

Commands for administrators:

  • /whereareyouadmin - Shows all administration commands for Where Are You.
  • /whereareyouadmin version - Checks for new updates.
  • /whereareyouadmin info <player> - Finds a player's current location.
  • You can also use /whereareyouadmin as /wrua.
